# Big-Mart-Sales-Dashboard


## Project Overview
This Power BI dashboard analyzes Big Mart sales data to identify trends, outlet performance, and product insights.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Total Sales Performance
- Outlet Type Analysis
- Item Category Analysis
- Location-wise Sales Comparison
- Interactive Dashboard Filters
